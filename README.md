# Extend Career - Frontend

AI-powered career platform for managing your entire job search.

## Features

- **CV Builder** - Create professional, ATS-friendly resumes with 30+ templates and live preview
- **Smart Job Matching** - AI-powered match scoring to find relevant opportunities
- **Email Management** - Centralized inbox for job search communications (Gmail/Outlook sync)
- **Interview Tracker** - Calendar integration with reminders and AI interview prep

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
- Supabase (auth & database)
- Stripe (payments)
- AI SDK (Anthropic)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint
